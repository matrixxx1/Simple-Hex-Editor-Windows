# Simple Hex Editor

## Short description

Inspect file bytes with a clear hex and text view.

## Full description

Simple Hex Editor is a Windows utility designed around a simple, local-first workflow. The initial app gives users a clear starting surface for open a file, inspect byte offsets, search values, and prepare edit notes.

## Feature bullets

- File byte inspection workflow
- Offset and hex preview concept
- Search surface
- Careful edit planning
