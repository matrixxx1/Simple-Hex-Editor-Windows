# Simple Hex Editor

## Short description

Inspect and edit local files in a fast, practical hex workflow with bookmarks, search, undo/redo, checksums, and export.

## Full description

Simple Hex Editor is a premium-ready local Windows utility for byte-level inspection and editing. It is designed for clarity and practical workflows: open any file, jump to exact offsets, search hex signatures, edit values safely, track changes, bookmark useful points, and generate hex dump exports.

The app includes:
- 16-byte hex layout with ASCII column
- Byte-by-byte editing with undo and redo
- Search and match navigation (`Find`, `Next`)
- Optional diff highlighting when comparing edits against the loaded baseline
- Bookmark offsets for quick revisit
- Hashing and checksum tools (CRC-32 and SHA-256)
- Full event logging and a built-in log viewer
- 15-day fully functional trial, then upgrade to unlock continuity
