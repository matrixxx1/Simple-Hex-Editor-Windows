# Store Submission Checklist

- Reserve the exact app name in Partner Center.
- Build MSIX package and confirm identity/reserved-name match.
- Generate final app icon and Store artwork.
- Capture required Store screenshots (landscape and dark mode variants).
- Capture app logs/troubleshooting behavior and test for 15-day trial expiration behavior.
- Confirm app icon is set via `SimpleHexEditor.ico` and app package metadata is correct.
- Upload package, StoreListing copy, screenshots, and PrivacyPolicy.txt in Partner Center.
- Confirm age rating and additional testing notes.
